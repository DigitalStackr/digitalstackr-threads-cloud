# Images Folder

Drop your real screenshots / dashboard pics / payout proofs in this folder.

**To add an image:**
1. Open this folder on github.com (the `images/` folder in your repo)
2. Click **"Add file" → "Upload files"** at the top
3. Drag the image from your phone/computer into the upload area
4. Scroll down → click **"Commit changes"**

**To use an image in a post:**
- In the "Add Post to Queue" workflow form, enter the filename (e.g. `dashboard_846.png`) in the `image_file` field

**Rules:**
- Real screenshots only. Never AI-generated.
- PNG, JPG, JPEG supported.
- Filenames should have no spaces — use underscores (e.g. `payout_screenshot_april.png`)
- This folder is committed to your repo. Old images can be deleted via the GitHub web UI when no longer needed.
